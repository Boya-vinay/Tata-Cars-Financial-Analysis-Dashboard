#  Tata Motors Financial Analysis

##  Before Dataset View (Understanding the Data)

The dataset used in this project contains financial and product-level information related to Tata Motors and its associated brands like Jaguar and Land Rover.

###  Initial Observations:

* Dataset contains **10,000+ records**
  
* Includes both **owned and manufactured vehicle data**
  
* Multiple financial columns such as revenue, profit, EBITDA, tax, and cost components
  
* Presence of **null/empty columns** (`extra_null_col1`, `extra_null_col2`)
  
* Some inconsistencies between **sales, cost, and profit values**
  
* Mixed data types (numbers stored as text, percentage formats, etc.)

---

##  Data Cleaning & Preparation

To make the dataset analysis-ready, the following steps were performed:

* Removed unnecessary **null columns**
* Handled missing values where required
* Corrected **data type issues** (currency, numbers, percentages)
* Verified and adjusted **revenue, cost, and profit relationships**
* Standardized column formats for accurate analysis

---

##  Project Overview

This project focuses on analyzing the financial performance of Tata Motors' car portfolio, including both **owned and manufactured vehicles**.

The goal is to generate insights into:

* Sales performance
* Revenue generation
* Profitability
* Cost structure

---

##  Objectives

* Analyze **sales and revenue trends**
* Compare **profit across different car brands**
* Understand impact of **manufacturing and operational costs**
* Evaluate key financial metrics like:

  * Net Income
  * EBITDA
  * Profit Margin

---

##  Analysis Performed

* Revenue vs Profit analysis
* Product-wise performance comparison
* Cost breakdown analysis
* KPI tracking (Revenue, Profit, Margin)
* Trend analysis using sales data

---

##  Tools & Technologies

* Power BI
* SQL
* Excel

---

##  Key Insights

* Premium brands (Jaguar, Land Rover) show **higher profit margins**
* Cost components significantly affect overall profitability
* Sales volume plays a major role in revenue generation
* Data cleaning was crucial to ensure accurate insights

---



## Conclusion

This project highlights the importance of data cleaning and structured analysis in deriving meaningful business insights from financial datasets.

# Tata-Cars-Financial-Analysis-Dashboard
This repository contains a financial analysis project on Tata Motors, focusing on car products that are owned or manufactured by the company. The project includes data cleaning, transformation, and interactive dashboard creation using Power BI and SQL to generate meaningful business insights and support decision-making.

High operational efficiency
